# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
숫자 맞추기 게임 프로그램

## 2. 핵심 기능 목록
(구현할 기능들을 순서대로 나열하세요.)
1.1부터 100 사이의 랜덤 숫자를 생성한다.

2.사용자가 숫자를 입력한다.

3.입력한 숫자가 정답보다 큰지, 작은지 비교하여 알려준다.

4.정답을 맞출 때까지 반복한다.

5.시도 횟수를 출력하고 게임을 종료한다   

## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)

**방법 1: 배열 사용 (기본)**
// 예시: 숫자 맞추기 게임에 필요한 변수들
int answer;     // 랜덤으로 생성된 정답 숫자
int guess;      // 사용자가 입력한 숫자
int count = 0;  // 시도 횟수

```

**방법 2: 구조체 사용 (선택)**
// 예시: 게임 정보를 구조체로 묶어서 관리

    int count;    // 시도 횟수
```
